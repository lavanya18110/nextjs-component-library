Build a personal portfolio website using Next.js (App Router), React, TypeScript, and Tailwind CSS.

Requirements:

1. Layout:
- Responsive design (mobile + desktop)
- Clean modern UI

2. Sections:

A. Hero Section:
- Name, title (Frontend Developer)
- Short tagline
- Buttons: "View Projects", "Download Resume"

B. About Section:
- Short introduction
- Skills summary

C. Skills Section:
- Display tech stack (React, Next.js, Angular, Node.js, Tailwind)
- Use icons or cards

D. Projects Section:
- Show at least 3 project cards
- Each card includes:
  - Title
  - Description
  - Tech stack
  - GitHub link
  - Live demo link

E. Contact Section:
- Email
- LinkedIn
- GitHub
- Simple contact form (optional)

3. Components:
- Navbar
- Footer
- ProjectCard
- SkillCard

4. Styling:
- Use Tailwind CSS
- Add hover effects and smooth transitions

5. Bonus:
- Dark mode toggle
- Smooth scrolling
- Animations

6. Code Quality:
- Use reusable components
- TypeScript for all components
- Clean folder structure

7. Deployment:
- Ready for deployment on Vercel
